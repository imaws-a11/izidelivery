# Meu Super App Delivery 🚚

Monorepo com os 3 aplicativos do sistema de delivery.

## Estrutura

| Pasta | Descrição | Deploy |
|---|---|---|
| `app-admin-completo` | Painel Administrativo | Vercel |
| `app-entregador-completo` | App do Motorista/Entregador | Vercel |
| `app-servicos-completo` | App de Serviços / Backend | Vercel |

## Como rodar localmente

Acesse cada pasta e rode:

```bash
npm install
npm run dev
```
